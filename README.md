Project Title
  <br>매출 영향 요인 정량화를 통한 P&L 효율 극대화 OLS 회귀분석
<br>Analysis Overview
  <br>총 매출에 가려진 거래당 순수 수익성 (Base Revenue)을 OLS 회귀분석으로 정량화하고, 수익성 중심 전략을 수립
  <br>데이터 특성 고려 시 발생 가능성이 높은 다중공선성 (Multicollinearity) 문제를 해결 하고 모델 안정성 확보 과정 포함
<br>Key Findings
  <br>평일 수익성 Base 우위 증명: 주말 (기준) 대비 평일 테이블 기본 매출 이 11,310원 더 높음 (P-value 0.021)
  <br>수익성 저해 요인 발견: 수량×중고생 상호작용 항 분석 결과, 중고생 유입 시 수량당 매출 효율이 973원 감소 함을 정량화 (P-value 0.000)
